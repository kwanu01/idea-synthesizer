# Idea Synthesizer

> 생각을 신디사이저처럼 조작하는 아이디어 워크스테이션.

### ▶ 바로 사용하기 — **https://idea-synthesizer-eight.vercel.app**

---

AI와 길게 대화해도 머리에 남는 구조는 없다. 좋은 문장, 버린 생각, 결정한 것이 한 스크롤에 뒤섞여 결국 처음부터 다시 훑게 된다. 문제는 아이디어가 부족한 게 아니라, **아이디어를 다룰 인터페이스가 없다**는 것이다.

Idea Synthesizer는 AI의 출력을 *소비*하는 대신 *조작*하게 만든다. 텍스트를 사고 단위 노드로 **분해**하고 → 케이블로 **연결**하고 → 변형 장치에 물려 **조작**하고 → 한 편의 글로 **종합**한다. 모듈러 신디사이저가 소리를 신호로 다루듯, 이 작품은 생각을 신호로 다룬다.

## 핵심

- **분해 → 연결 → 조작 → 종합.** 사용자가 손을 대지 않으면 아무것도 나오지 않는다. 인지적 개입이 구조로 강제된다.
- **에이전트.** 실제 인물의 글로 학습한 페르소나를 갈아 끼우면, 같은 회로라도 결과의 관점·어휘가 통째로 바뀐다. 내 글로 새 에이전트를 직접 구울 수도 있다.
- **이펙터 6종.** 관점 · 반박 · 추론 · 제약 · 추상 · 연결. 케이블의 관계가 변형 방향으로 주입된다.

## 사용법

1. 아이디어·메모·AI와 나눈 대화를 입력하고 **Analyze** — 문장이 노드로 쪼개진다. (바로 써보려면 **Load Sample**)
2. **Auto Patch** — 노드가 이펙터를 거쳐 출력까지 케이블로 이어진다.
3. 이펙터 **노브를 드래그** — 변형 방식이 바뀌고, 손을 떼면 결과가 다시 만들어진다.
4. **STATEMENT** — 회로가 출력에 닿으면 한 편의 글로 자동 종합된다.
5. 왼쪽 위 **에이전트**를 바꾸면 같은 회로라도 결과의 관점이 바뀐다.
